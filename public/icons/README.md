# 모바일 아이콘 파일 넣는 곳

이 폴더에 아래 3개 PNG 파일을 넣으면 모바일 홈 화면 아이콘이 동작합니다.

| 파일명              | 크기      | 용도                                  |
|---------------------|-----------|---------------------------------------|
| icon-192.png        | 192×192   | Android 홈 화면 / PWA 기본 아이콘     |
| icon-512.png        | 512×512   | Android 스플래시 / 고해상도 기기      |
| apple-touch-icon.png| 180×180   | iPhone/iPad 홈 화면 추가              |

## 만드는 방법 (무료)

1. https://favicon.io/favicon-generator/ 에 접속
2. 텍스트: "세" 또는 "S", 배경색: #c8a96e (골드), 글자색: #ffffff
3. Download → ZIP 안에 있는 파일을 이 폴더에 저장

또는 디자인 파일(로고 등)이 있으면 PNG로 내보내 이 폴더에 저장하면 됩니다.

## 현재 상태

아이콘 파일이 없으면 브라우저 기본 아이콘(빈 페이지 아이콘)이 표시됩니다.
파일을 추가한 뒤 Vercel에 다시 배포하면 적용됩니다.
